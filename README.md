A simple, no dependency static file server that runs on NodeJS.  
Start via `node staticFileServer.js` and point your browser to http://localhost:8080/path/to/yourFile.ext.  
Create a custom `config.js` file and start the server via `node staticFileServer.js path/to/your/config.js` to overwrite the default config values from [config.sample.js](config.sample.js).
